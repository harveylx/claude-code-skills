Open Source Replacer

ln-645 L3 Worker v1.0.0

Overview

Purpose: Discover custom modules, analyze their purpose, find battle-tested OSS replacements via MCP Research

Type: L3 Worker (invoked by ln-640-pattern-evolution-auditor)

Checks: Module discovery, pre-classification, goal extraction, alternative search, security gate, evaluation, migration plan

Key feature: Goal-based deep analysis (not pattern matching). Reads code to understand PURPOSE, then searches for packages that solve the same problem.
